Question-1: What are the different ways to select an element in the DOM?

In JavaScript, elements in the DOM can be selected using methods such as document.getElementById("myId") to select by ID, document.getElementsByClassName("myClass") to select by class, document.getElementsByTagName("div") to select by tag name, document.querySelector(".myClass") to select the first matching element, and document.querySelectorAll(".myClass") to select all matching elements.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML returns or sets the HTML content inside an element, preserving HTML tags and including hidden elements. innerText returns or sets only the visible text inside an element and does not include hidden elements. textContent returns or sets all text content inside an element, including hidden elements, but does not preserve HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is attached to a parent element, and it handles events from child elements using event bubbling. This allows for more efficient event handling, especially for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event triggered on a child element bubbles up through its parent elements, allowing each parent to handle the event. It can be stopped using event.stopPropagation() to prevent the event from reaching the parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement("div"). To add the created element to the DOM, use element.appendChild(newElement). To remove an element, use element.remove() or parentElement.removeChild(element).
